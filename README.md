## ClockChord
<a href="https://kamide-akiyoshi.github.io/clockchord/"><img src="ClockChord.png"/></a>

Chord-visualizable Musical Chord Instrument on the Circle-of-fifths Clock for web browser,
with MIDI(SMF) player, and embedded player linked to [Songle Widget](https://widget.songle.jp/).
